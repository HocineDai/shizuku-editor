## 🚀 What it is

shizuku-editor is a desktop writing app for authors. It helps you keep chapters in order, split work into scenes, track versions, and export your manuscript in common file formats.

It runs on Windows, macOS, and Linux.

## 💻 Windows download and setup

1. Open the [releases page](https://github.com/HocineDai/shizuku-editor/releases).
2. Find the latest version at the top.
3. Under **Assets**, choose the Windows file.
4. Download and run that file.
5. If Windows asks for permission, choose **Yes**.

If you use the app often, keep the downloaded file in a folder you can find again.

## 📦 What you can do

## 🖱️ First run

When you open shizuku-editor for the first time:

1. Start the app from the file you downloaded.
2. Wait for the window to open.
3. Create a new project or open an existing one.
4. Add your title, chapters, and scenes.
5. Save your work as you go.

If you already have writing in another format, you can bring it into a new project and sort it from there.

## ✍️ Typical workflow

A simple way to use the app:

1. Create a manuscript project.
2. Add a chapter.
3. Add scenes inside that chapter.
4. Write in the editor.
5. Reorder scenes if the story changes.
6. Check older versions when you want to compare edits.
7. Export the final draft when you are ready to share it.

This setup works well for novels, short stories, and long-form drafts.
